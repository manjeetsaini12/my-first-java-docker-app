# My First Java Docker App

A simple Java application containerized using Docker.

## 🛠 Tech Stack

* Java
* Docker

## ▶️ Run Locally

Build the image:

```bash
docker build -t java-docker-app .
```

Run the container:

```bash
docker run java-docker-app
```

## 📌 Output

Displays a simple message in the terminal.
